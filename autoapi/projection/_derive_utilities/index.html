Converts a projected numpy array (derived from a data-frame) back into data-frame format with row.names

Parameters

  • unprojected – the original data-frame from which the projection was derived

  • projected – the projection derived from unprojected

  • feature_prefix – a prefix to use in feature names to become feature_prefix0, feature_prefix1 etc.

Returns

returns a data-frame with identical row order and sze to df_orig but with embeddings from projection
